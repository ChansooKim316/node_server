### 작동순서  
#
#### 1. 클러스터들 시작 (index.js 가 마스터 역할)
#### 2. 리퀘스트로 부터 헤더들 저장
#### 3. 리퀘스트를 분배 (헤더 첫줄 ~ '\r\n\r\n' 까지 합쳐 리턴)
#### 4. 가상호스트, 메소드 분기
#### 5. 리스폰스를 텍스트로 만들어 변수에 할당 
- 리다이렉션 
- 쿼리스트링 또는 바디의 데이터들을 처리
- 파일요청
- 캐시컨트롤

#### 6. 리스폰스 보내기
